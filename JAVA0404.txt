Final 키워드의 용도
  - final 필드 : 수정 불가 필드
  - final 클래스 : 부모로 사용 불가한 클래스
  - fianl 메소드 : 자식이 재정의할 수 없는 메소드 (Override 불가)

상속할 수 없는 final 클래스
  -  자식 클래스 만들지 못하도록 final 클래스로 생성
	- public final class 클래스 (...)
	- public final class String (...)
	- public 

오버라이딩 불가한 final 메소드
  - 자식 클래스가 재정의 못하도록 부모 클래스의 메소드를 final로 생성

Overload
 - 하나의 메소드를 여러개로 만들어서 사용
	m(1){}
	m(1,2){}
	m("ffd"){}

Override
 - 부모에서 정의된 객체를 자식 객체에서 사용

protected 접근 제한자
 - 상속과 관련된 접근 제한자
	- 같은 패키지 : default와 동일
	- 다른 패키지 : 자식 클래스만 접근 허용

다형성 (Polymorphism)
 - 같은 타입이지만 실행 결과가 다양한 객체 대입(이용) 가능한 성질
	- 부모타입에는 모든 자식 객체가 대입 가능
	- 자식 타입은 부모 타입으로 자동 타입 변환

 - 효과 : 객체 부품화 가능

 - 변환 후에는 부모 클래스 멤버만 접근 가능
 - 부모 타입으로 자동 타입 변환
 - 재정의된 메소드(Override)

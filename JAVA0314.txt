연산자

단항  > 이항  -x + 6
곱셈, 나눗셈 > 덧셈, 뺄셈  x + 5 * y
산술 > 비교 x + 3 > y - 1
비교 > 논리 x > 5 && x < 10
대입은 맨 마지막에 한다.
괄호는 최 우선 순위

int val1 = 1;
int val2 = 2;
int val3 = 3;
int result = val1 + val2 * val 3;
	= 7;
int result = (val1 + val2) * val 3;
	= 9;

단항 연산자 - 피연산자가 1개인 연산자

-단항 연산자의 종류
부호 연산자 : +, -
boolean 타입과 char 타입을 제외한 기본 타입에 사용 가능

증감연산자 : ++, --
변수의 값을 1증가 시키거나(++) 1감소(--)시키는 연산자

증감 연산자가 변수 뒤에 있으면 다른 연산자 먼저 처리 후 증감 연산자를 계산한다.

논리 부정 연산자 : !
blooean type 에만 사용가능.

비트 반전 연산자 : ~ (틸트)
byte, short, int, long 타입만 피연산자가 될 수 있다. char(x)
비트 값을 반전 시킨다. (0 => 1 , 1 => 0)

부호 비트인 최상위 비트까지 반전 되므로 부호의 반대값이 산출된다.
byte v1 = 10;
int v2 = ~v1;  = -11
int v2 = ~v1 + 1; = -10


런타임시 입력된 값을 계산하여야 할 경우
 - 산술 연산자를 직접 사용하는 것 보다는 메소드를 작성하여
   오버플로우를 미리 조사하고 예외처리를 한다.

산술 연산자
결과값 산출할 때 Overflow 주의

정확한 계산은 정수를 사용

Nan 과 infinity 연산은 주의할 것
5 / 0 --- 5 % 0

문자열 연산자
피연산자 중 문자열이 있으면 문자열로 결합

비교 연산자
==, !=, <, >, <=, >= 
true, false 로 출력

동등 비교 연산자는 모든 타입에 사용
크기 비교 연산자는 boolean타입 제외한 모든 기본 타입에 사용
